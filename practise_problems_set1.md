# Python Practice Problems 🐍

This repository contains **beginner-friendly Python practice problems** to strengthen fundamental programming concepts.

Topics covered:

* Variables
* Operators
* Control Flow (if, else, elif)
* Loops

---

# 1️⃣ Variables – 10 Practice Questions

## Very Easy

1. Create a variable `name` and store your name in it. Print the variable.

2. Store your `age` in a variable and print:

```
My age is 20
```

*(Replace 20 with your value.)*

3. Create two variables:

```python
a = 5
b = 10
```

Print both variables.

---

## Easy

4. Store your city name in a variable and print:

```
I live in Chennai
```

5. Create a variable:

```python
price = 100
```

Print the value multiplied by 2.

6. Store your favorite programming language in a variable and print it.

---

## Slightly Challenging

7. Create variables:

```python
x = 10
y = 20
```

Swap the values.

8. Create a variable:

```python
pi = 3.14
```

Print:

```
Value of PI is 3.14
```

9. Store:

```python
length = 10
width = 5
```

Print the **area of rectangle**.

10. Create 3 variables `name`, `age`, `course` and print them in one sentence.

Example output:

```
My name is John, I am 20 years old and learning Python
```
